About This App

My Weather App is a dynamic and user-friendly web application that provides real-time weather information for cities around the world. Using the power of the WeatherAPI, this app delivers up-to-date details such as:

  • 🌡️ Temperature in Celsius and Fahrenheit
  • 💨 Wind speed and direction
  • 💧 Humidity, visibility, and cloud cover
  • 🌡️ Feels like temperature
  • ☔ Precipitation levels
  • 🌤️ Weather conditions with visual icons

The app also features a search functionality and dropdown for common cities, making it easy to check weather conditions anytime, anywhere.

User Guide

This guide will help you use My Weather App efficiently to check weather conditions for any city:

  • 🔍 Search for a City: Type the city name in the search box and click the “Search” button.
  • 📋 Dropdown Menu: Quickly select popular cities from the “How To Use?” dropdown menu.
  • 🌤️ Weather Details: View temperature, wind, humidity, cloud coverage, visibility, precipitation, and feels like temperature in the cards.
  • 📊 Common Places Table: Scroll down to see live weather data for commonly searched cities.
  • 🖼️ Weather Icons: Visual representation of current weather is displayed next to the city name.

Enjoy exploring weather data from around the world in a simple, interactive way!
